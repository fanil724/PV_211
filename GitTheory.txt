VCS - Version Control System
SVN, Subversion, Mercury, .... Git/GitHub
Git != GitHub
Git - система контроля версий.
GitHub - облачное хранилище.

Repository - это хранилище исходных кодов.
Local/Remote repository
fetch - скачивает изменения с удаленного репозитория в локальный, и НЕ распаковывает их.
pull  - скачивает и распаковывает изменения с удаленного репозитория в локальный.
push  - загружает изменения с локального репозитория на удаленный репозиторий.

commit - это контрольная точка, которая содержит снимки всех нужных файлов репозитория.
		 Чтобы создать commit обязательно нужно написать commit-message

branch - это последовательность commit-ов
master - это главная ветка в любом репозитории
merge - слияние (веток)
pull-request - запрос на слиние веток
